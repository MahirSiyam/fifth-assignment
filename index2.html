Question-1: What are the different ways to select an element in the DOM?

Ans: document.getElementById('Id') selects an element by its ID, while document.querySelector('.Class') selects the first element with a specific class.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: innerHTML includes HTML content and can render tags, innerText shows only visible text without hidden elements, and textContent displays all text content including hidden elements without rendering HTML tags.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique in JavaScript where you add a single event listener to a parent element to handle events for all of its child elements. It leverages event bubbling to manage dynamic content efficiently and reduce the number of event listeners.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a behavior in the DOM where an event triggered on a child element propagates up through its ancestors in the hierarchy. This allows parent elements to handle events that occur on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: To create an element, use document.createElement(tagName). To add it, use parentElement.appendChild(newElement). To remove it, use parentElement.removeChild(childElement).
